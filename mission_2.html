Context de la mission

Dans cette mission, nous avions comme objectif de commencer à exploiter la même base de données qu’à la mission 1, la base de données de l’entreprises Kdou. Cependant ici nous devions utiliser le logiciel Low Code Acces pour analyser la base et obtenir des indicateurs intéressants à l’aide du mode de création graphique QBE (Query By Example),ayant une première introduction au SQL. La base de données est composée de 6 relations : Fournisseur, Produit , Categorie, Clientele, Commande, DetailCommande.

Indicateur intéressant relever dans la base

Produit Allemand et Italiens inférieurs à 50€

Quels sont les produits d’Italie ou d’Allemagne vendus 50€ ou moins ?

Nous avons cherché les produits provenant d’Italie ou d’Allemagne (ayant l’adresse du fournisseur dans un des deux pays) ayant un prix de vente unitaire à 50€ ou moins.

Voici le code généré par cette requête.
SELECT DISTINCT Produit.NomProd, Produit.PrixUnit, Fournisseur.Contact, Fournisseur.Ville, Fournisseur.Pays FROM Fournisseur INNER JOIN (Produit INNER JOIN DetailCommande ON Produit.RefProd = DetailCommande.RefProd) ON Fournisseur.NoFour = Produit.NoFour WHERE (((Fournisseur.Pays)="Allemagne" Or (Fournisseur.Pays)="Italie")) ORDER BY Produit.PrixUnit;

Nous avons cherché les produits de type viandes ayant été commandés par un client ayant son adresse sur Paris.

Voici le code généré par cette requête.
SELECT DISTINCT Produit.NomProd, Categorie.NomCateg, Client.VilleCli FROM Categorie INNER JOIN ((Client INNER JOIN Commande ON Client.CodeCli = Commande.CodeCli) INNER JOIN (Produit INNER JOIN DetailCommande ON Produit.RefProd = DetailCommande.RefProd) ON Commande.NoCom = DetailCommande.NoCom) ON Categorie.CodeCateg = Produit.CodeCateg WHERE (((Categorie.NomCateg)="Viandes") AND ((Client.VilleCli)="Paris"));

Commande du client PAVBO

Quels produits (nom, catégorie et prix) a commandé le client Mikal Pavbo ?

Nous avons cherché toutes les commandes qu’à fait le client avec le code client PAVBO, ce dernier étant celui du client Mikal Pavbo.

Voici le code généré par cette requête.
SELECT DISTINCT Produit.NomProd, Categorie.NomCateg, Produit.PrixUnit FROM Categorie INNER JOIN (Client INNER JOIN (Commande INNER JOIN (Produit INNER JOIN DetailCommande ON Produit.RefProd = DetailCommande.RefProd) ON Commande.NoCom = DetailCommande.NoCom) ON Client.CodeCli = Commande.CodeCli) ON Categorie.CodeCateg = Produit.CodeCateg WHERE (((Client.CodeCli)="PAVBO"))

Prix moyen, maximum et minimum des produits

Donner le prix moyen de vente des produits, le prix minimum et le prix maximum.

Tableau résultat de la requête
MoyenneDePrixUnit MinDePrixUnit MaxDePrixUnit
33.54€ 2.08€ 320.00€

Nous avons fait un calcul sur la base de tous les produits de la base pour avoir la moyenne de prix, le maximum et le mimum.

Voici le code généré par cette requête.
SELECT Avg(Produit.PrixUnit) AS MoyenneDePrixUnit, Min(Produit.PrixUnit) AS MinDePrixUnit, Max(Produit.PrixUnit) AS MaxDePrixUnit FROM Produit;
